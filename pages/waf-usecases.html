Volume & Traffic Anomalies

UC1: Block Volume Anomaly

Detect statistical spikes in deny/block volume using baseline comparison. Indicates scanning, exploit attempts, or DDoS activity.

MEDIUM T1190 Every 5 min

UC2: Single Source Flood

Single IP generating excessive requests. Indicates brute force, credential stuffing, or DoS from single attacker.

HIGH T1498.001 Every 5 min

UC5: Distributed Low-and-Slow DDoS

Many IPs each sending few requests. Classic botnet pattern that evades single-source detection.

HIGH T1498 Every 5 min

Application Attacks

UC3: Authentication Endpoint Attack

Attacks targeting login/auth endpoints. Brute force and credential stuffing campaigns.

HIGH T1110 Every 5 min

UC4: SQL Injection / XSS Campaign

CRITICAL: Database and session compromise risk. A single successful SQLi can compromise your entire database.

CRITICAL T1190 Every 5 min

UC9: API Endpoint Abuse

Excessive API requests indicating data scraping, enumeration, or abuse of API endpoints.

CRITICAL T1190 Every 5 min

UC10: Path Traversal Attempts

Directory traversal attacks using ../ patterns to access sensitive files.

HIGH T1083 Every 5 min

Reconnaissance & Evasion

UC6: Geographic Anomaly
