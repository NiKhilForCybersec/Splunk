WAF-UC8: Bot/Scraper Detection

Detect automated scrapers and bots by analyzing behavioral patterns: high request rates, sequential URL access, lack of JavaScript execution, suspicious user agents, and missing typical browser headers.

Severity MEDIUM
MITRE T1595
Schedule Every 15 min
Time Range Last 30 min

Step 1: Decision Tree (Detection Logic)

flowchart TD
    START[Aggregate requests by source IP over 30 min] --> A[Calculate behavioral indicators]
    A --> B[Count total requests]
    B --> C[Count unique URIs accessed]
    C --> D[Check user agent patterns]
    D --> E{Is request count GT 500}
    E -->|No| F{Is request count GT 100 with suspicious UA}
    F -->|No| LOW[Below bot threshold - normal traffic]
    F -->|Yes| SUSPICIOUS[Suspicious - low volume but bot UA]
    E -->|Yes| G{Calculate URI diversity ratio}
    G --> G1[ratio = unique_uris divided by total_requests]
    G1 --> H{Is ratio GT 0.8 - accessing many unique pages}
    H -->|Yes| SCRAPER[SCRAPER PATTERN - Crawling many pages]
    H -->|No| I{Is ratio LT 0.1 - hitting same pages repeatedly}
    I -->|Yes| HAMMER[HAMMERING - Same endpoint repeatedly]
    I -->|No| MIXED[Mixed pattern - needs investigation]
    SCRAPER --> J{Is user agent a known bot}
    J -->|Yes| KNOWN_BOT[Known Bot - check if legitimate]
    J -->|No| STEALTH[Stealth Scraper - hiding identity]
    HAMMER --> HIGH1[HIGH - Possible attack or abuse]
    STEALTH --> HIGH1
    KNOWN_BOT --> K{Is bot in legitimate bots list}
    K -->|Yes| LEGIT[Legitimate bot - Googlebot etc]
    K -->|No| BAD_BOT[Bad Bot - unknown automated access]
    BAD_BOT --> MED[MEDIUM severity]
    SUSPICIOUS --> MED
    MIXED --> MED
    HIGH1 --> CREATE[Create Notable Event]
    MED --> CREATE

Step A: Build SPL in Search First

Where: Splunk Search ‚Üí Build and test SPL

Bot Detection with Behavioral Analysis

index=akamai sourcetype=akamai:siem earliest=-30m
| spath
| eval src_ip=coalesce(src_ip,'attackData.clientIP')
| eval uri=coalesce(uri,'httpMessage.requestUri')
| eval vhost=coalesce(vhost,'httpMessage.requestHeaders.Host')
| eval user_agent=coalesce(http_user_agent,'httpMessage.requestHeaders.User-Agent')
| stats count as total_requests
        dc(uri) as unique_uris
        values(user_agent) as user_agents
        dc(user_agent) as ua_count
        BY src_ip, vhost
| where total_requests > 100
| eval uri_diversity = round(unique_uris / total_requests, 2)
| eval is_known_good_bot = if(match(mvjoin(user_agents,","), "(?i)(googlebot|bingbot|slurp|duckduckbot|facebookexternalhit|twitterbot|linkedinbot)"), "Yes", "No")
| eval is_scraper_ua = if(match(mvjoin(user_agents,","), "(?i)(python|curl|wget|scrapy|httpclient|java\/|go-http|ruby|perl|php|libwww)"), "Yes", "No")
| eval is_headless = if(match(mvjoin(user_agents,","), "(?i)(headless|phantom|puppeteer|selenium|playwright)"), "Yes", "No")
| eval is_empty_ua = if(ua_count=0 OR match(mvjoin(user_agents,","), "^.{0,5}$"), "Yes", "No")
| eval bot_type = case(
    is_known_good_bot="Yes", "Legitimate Search Bot",
    is_headless="Yes", "Headless Browser (Suspicious)",
    is_scraper_ua="Yes", "Known Scraper Tool",
    is_empty_ua="Yes", "Empty/Suspicious UA",
    uri_diversity > 0.8 AND total_requests > 500, "Stealth Scraper (high diversity)",
    uri_diversity < 0.1 AND total_requests > 500, "Endpoint Hammering",
    1=1, "Unknown Automated"
)
| eval behavior = case(
    uri_diversity > 0.8, "Crawling (many unique pages)",
    uri_diversity < 0.1, "Hammering (same pages)",
    1=1, "Mixed"
)
| eval severity = case(
    is_known_good_bot="Yes", "info",
    is_headless="Yes" AND total_requests > 500, "high",
    is_scraper_ua="Yes" AND total_requests > 500, "high",
    is_empty_ua="Yes", "high",
    total_requests > 1000, "medium",
    1=1, "low"
)
| where severity!="info" AND severity!="low"
| table src_ip vhost total_requests unique_uris uri_diversity bot_type behavior user_agents severity

Internal tools Corporate IP ranges, internal automation Exclude internal IP ranges from detection
Mobile apps Custom mobile app user agents Identify your mobile app UA patterns; exclude from bot detection
