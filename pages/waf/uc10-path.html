WAF-UC10: Path Traversal Attempts

Directory traversal attacks using ../ patterns to access files outside the web root. Attackers attempt to read sensitive files like /etc/passwd, configuration files, or source code.

Severity HIGH
MITRE T1083
Schedule Every 5 min
Time Range Last 10 min

Path Traversal Pattern Reference

Common Traversal Patterns and Encoding Evasions

PatternExampleEvasion Technique
Basic ../../../etc/passwd Simple directory traversal
URL Encoded %2e%2e%2f Encode dots and slashes
Double Encoded %252e%252e%252f Bypass WAF that decodes once
Unicode ..%c0%af Unicode encoding of /
Windows Style ..\..\..\ Backslash for Windows
Null Byte ../../../etc/passwd%00.jpg Truncate file extension check

High-Value Target Files

FileOSWhy TargetedSeverity
/etc/passwd Linux User enumeration, system info CRITICAL
/etc/shadow Linux Password hashes CRITICAL
.env, config.php Any Database credentials, API keys HIGH
web.config, app.config Windows/.NET Connection strings, keys HIGH
wp-config.php WordPress WordPress DB credentials HIGH
boot.ini, win.ini Windows System configuration MEDIUM

Step A: Build SPL in Search First

Where: Splunk Search → Build and test SPL

Path Traversal Detection

index=akamai sourcetype=akamai:siem earliest=-10m
| spath
| eval waf_action=coalesce(action, mvindex('attackData.rules{}.ruleAction',0))
| eval src_ip=coalesce(src_ip,'attackData.clientIP')
| eval uri=coalesce(uri,'httpMessage.requestUri')
| eval vhost=coalesce(vhost,'httpMessage.requestHeaders.Host')
| where waf_action="deny" OR waf_action="block"
| where match(uri, "(?i)(\.\.\/|\.\.\\\\|%2e%2e|%252e|%c0%ae|%c0%af|%00)")
| eval traversal_type = case(
    match(uri, "%252e"), "Double URL Encoded",
    match(uri, "%c0%a"), "Unicode Encoded",
    match(uri, "%2e%2e"), "URL Encoded",
    match(uri, "%00"), "Null Byte Injection",
    match(uri, "\.\.\\\\"), "Windows Backslash",
    1=1, "Basic ../"
)
| eval target_file = case(
    match(uri, "(?i)(passwd|shadow)"), "Linux Credentials",
    match(uri, "(?i)(\.env|config\.php|wp-config|database\.yml)"), "Configuration File",
    match(uri, "(?i)(web\.config|app\.config|appsettings)"), "Windows/.NET Config",
    match(uri, "(?i)(\.git|\.svn|\.hg)"), "Version Control",
    match(uri, "(?i)(boot\.ini|win\.ini)"), "Windows System",
    1=1, "Other"
)
| stats count as attempts
        dc(uri) as unique_paths
        values(traversal_type) as encoding_types
        values(target_file) as target_categories
        values(uri) as sample_uris
        BY src_ip, vhost
| eval severity = case(
    match(mvjoin(target_categories,","), "Credentials") AND attempts > 5, "critical",
    match(mvjoin(target_categories,","), "Credentials"), "high",
    match(mvjoin(target_categories,","), "Configuration") AND attempts > 10, "critical",
    match(mvjoin(target_categories,","), "Configuration"), "high",
    attempts > 20, "high",
    1=1, "medium"
)
| table src_ip vhost attempts unique_paths encoding_types target_categories sample_uris severity
