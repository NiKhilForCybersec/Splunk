MFA Fatigue Attack Chain

1. Credential Theft

Attacker obtains password via phishing, credential stuffing, or dark web purchase

2. Authentication Attempt

Attacker enters valid credentials, triggering MFA challenge

3. User Denies

User receives unexpected push notification and denies it

4. Repeated Push Bombing

Attacker triggers 5, 10, 20+ prompts rapidly to exhaust user

5. User Capitulates

Exhausted or confused user approves → Account Compromised

Detection Goal
Detect and alert BEFORE step 5. 5+ denials in 10 minutes triggers HIGH alert. If followed by success = CRITICAL (likely compromise).
