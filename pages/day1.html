Day 1 Data Validation

Run these queries immediately to validate your data foundation:

Data Validation Queries
``` 1. Check Akamai data exists and is recent
index=akamai earliest=-24h | stats count by sourcetype
| append [| makeresults | eval sourcetype="NO DATA" | where count=0]

``` 2. Check CyberArk data exists and is recent
index=cyberark earliest=-24h | stats count by sourcetype

``` 3. Check Entra ID data exists and is recent
index=azure sourcetype=azure:aad:* earliest=-24h | stats count by sourcetype

``` 4. Find all available indexes (understand the environment)
| eventcount summarize=false index=* | dedup index | table index

``` 5. Check data freshness (is data arriving on time?)
index=akamai earliest=-1h
| stats max(_time) as latest_event
| eval lag_minutes = round((now() - latest_event) / 60, 1)
| eval status = if(lag_minutes > 30, "‚ö†Ô∏è DELAYED", "‚úÖ OK")

``` 6. Check field extraction is working
index=akamai earliest=-1h | head 1
| table src_ip, dest_ip, action, http_uri, http_method
``` All fields should have values, not null
