Project Scope

The WAF/DDoS detection project delivers 8 use cases that provide visibility into:

  • Web Application Attacks — SQLi, XSS, command injection, file inclusion
  • DDoS Attacks — Volumetric, application-layer, distributed attacks
  • Credential Attacks — Brute force, credential stuffing on auth endpoints
  • Bot Activity — Malicious bots, scraping, automated attacks
  • Geographic Anomalies — Unexpected access patterns by location

Tier Definitions

Tier 1 — Foundation Detections

  • High confidence, low false positive rate
  • Detect obvious/volumetric attacks
  • Deploy first, establish baseline
  • Use cases: UC1-UC4

Tier 2 — Advanced Detections

  • Higher complexity, requires tuning
  • Detect evasive/sophisticated attacks
  • Deploy after Tier 1 is stable
  • Use cases: UC5-UC8

Akamai Integration Details

Splunk Add-on for Akamai
All WAF detections require the Splunk TA for Akamai (TA-akamai) to be installed and configured. This provides SIEM API integration, field extractions, and CIM mapping.
