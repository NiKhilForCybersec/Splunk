Glossary

Terminology, acronyms, and definitions for detection engineering

A B C D E F H I K L M O P R S T W

A

Term Definition
Akamai Content delivery network (CDN) and cloud security provider. Provides WAF (Kona Site Defender), Bot Manager, and DDoS protection services. The confirmed WAF vendor for Sobeys.
Alert A notification generated when a detection search finds events matching specified criteria. Can trigger email, ticket creation, or other actions.
Allowlist A list of known-good entities (IPs, users, applications) that should be excluded from detection logic to reduce false positives.
ATO (Account Takeover) An attack where adversaries gain unauthorized access to a user account, typically through credential theft, phishing, or MFA bypass.
ATT&CK (MITRE) Adversarial Tactics, Techniques, and Common Knowledge — a framework cataloging attacker behaviors. Used to map detections to known threat techniques.

B

Term Definition
Baseline A statistical measure of normal behavior (mean, standard deviation, percentile) calculated from historical data. Used to determine anomaly thresholds.
Bot Automated software that performs tasks over the internet. Can be legitimate (search crawlers) or malicious (scrapers, credential stuffers).
Brute Force An attack that tries many passwords or passphrases hoping one is correct. MITRE ATT&CK T1110.

C

Term Definition
CIAM Customer Identity and Access Management — systems managing customer (not employee) authentication. SAP CDC (Gigya) is Sobeys' CIAM platform.
CIM (Common Information Model) Splunk's data normalization framework. Maps vendor-specific fields to standard field names for cross-source analysis.
Credential Stuffing Attack using stolen username/password pairs from breaches to attempt logins. Relies on password reuse across sites.
CyberArk Privileged access management (PAM) vendor. Provides MFA and session management for privileged users. Primary MFA source for privileged accounts at Sobeys.

D

Term Definition
DDoS Distributed Denial of Service — an attack using many sources to overwhelm a target with traffic, causing service disruption. MITRE ATT&CK T1498, T1499.
Detection A search or rule designed to identify security events of interest. In Splunk, typically a scheduled saved search with alert actions.
Detection Engineering The practice of designing, implementing, testing, and maintaining security detections in SIEM platforms.

E

Term Definition
Entra ID Microsoft's identity and access management service (formerly Azure Active Directory). Provides SSO, MFA, and conditional access for corporate users.
ES (Enterprise Security) Splunk's security operations platform built on Splunk Enterprise. Provides Notable Events, Risk-Based Alerting, and MITRE mapping.

F

Term Definition
False Negative (FN) A missed detection — when malicious activity occurs but the detection fails to alert. Reduces detection coverage.
False Positive (FP) A false alarm — when a detection fires on legitimate activity. Causes alert fatigue and wastes analyst time.
FP Rate The percentage of alerts that are false positives. Target: <10% for production detections.

H

Term Definition
HEC (HTTP Event Collector) Splunk's HTTP-based data ingestion endpoint. Allows sending events via HTTP/HTTPS without deploying a forwarder.
Hypercare The period immediately after go-live with elevated monitoring and support. Typically 1-2 weeks for rapid tuning.

I

Term Definition
Impossible Travel A detection pattern identifying logins from geographically distant locations in an impossible timeframe (e.g., 1000km in 5 minutes).
Index A Splunk data repository. Events are stored in indexes with specific retention and access policies.
IoC (Indicator of Compromise) Observable artifacts (IPs, domains, hashes) indicating potential malicious activity.

K

Term Definition
Kona Site Defender (KSD) Akamai's web application firewall product. Provides OWASP protection, custom rules, and rate limiting.
KT (Knowledge Transfer) The process of documenting and transferring knowledge to the client team at project completion.

L

Term Definition
Latency The delay between when an event occurs and when it appears in Splunk. Affects detection timeliness.
LFI (Local File Inclusion) A web attack exploiting vulnerable include mechanisms to read local files. Often targets /etc/passwd or config files.
Lookup A Splunk feature that enriches events with data from CSV or KV store tables. Used for allowlists, user enrichment, etc.
Low-and-Slow An attack technique using slow, distributed activity to evade detection thresholds.

M

Term Definition
MFA (Multi-Factor Authentication) Authentication requiring multiple factors (something you know, have, or are). Critical control against credential theft.
MFA Fatigue An attack sending repeated MFA prompts to exhaust user patience, hoping they approve to stop notifications. MITRE ATT&CK T1621.
MTTD (Mean Time to Detect) Average time between attack occurrence and detection. Key SOC metric.
MTTR (Mean Time to Respond) Average time between detection and containment. Key SOC efficiency metric.

O

Term Definition
OWASP Open Web Application Security Project — a nonprofit producing security standards and tools. OWASP Top 10 lists common web vulnerabilities.

P

Term Definition
PAM (Privileged Access Management) Systems controlling access to privileged accounts. CyberArk is Sobeys' PAM solution.
PIPEDA Personal Information Protection and Electronic Documents Act — Canada's federal privacy law. Governs how private sector handles personal information.
Props.conf Splunk configuration file defining how data is indexed and extracted. Controls timestamp, field extraction, and transforms.
Push Bombing See MFA Fatigue. Also called "prompt bombing."

R

Term Definition
RCE (Remote Code Execution) A vulnerability allowing attackers to run arbitrary code on target systems. Critical severity.
Runbook A documented procedure for SOC analysts to follow when responding to a specific alert type.

S

Term Definition
SAP CDC (Gigya) SAP Customer Data Cloud (formerly Gigya) — Customer identity platform for consumer-facing applications.
SIEM Security Information and Event Management — platform aggregating logs for security monitoring. Splunk is Sobeys' SIEM.
SOC (Security Operations Center) The team responsible for monitoring and responding to security alerts.
Sourcetype Splunk's classification of data format. Determines parsing rules and field extractions.
SPL (Search Processing Language) Splunk's query language for searching, filtering, and analyzing data.
SQLi (SQL Injection) An attack injecting malicious SQL code to manipulate databases. OWASP Top 10 vulnerability.
Standard Deviation (σ) A statistical measure of variation from the mean. Used in threshold calculation: mean + 2σ catches ~95% of anomalies.

T

Term Definition
TA (Technology Add-on) A Splunk app providing data inputs, field extractions, and CIM mapping for specific technologies.
Threshold The boundary value that triggers an alert when exceeded. Can be static (fixed number) or dynamic (calculated from baseline).
TP Rate (True Positive Rate) The percentage of alerts that are genuine security events. Target: >80% for production detections.
Tuning The process of adjusting detection logic, thresholds, or allowlists to improve accuracy.

W

Term Definition
WAF (Web Application Firewall) A security control filtering HTTP traffic to protect web applications from attacks. Akamai KSD is Sobeys' WAF.
XSS (Cross-Site Scripting) An attack injecting malicious scripts into web pages viewed by other users. Can steal sessions or credentials.
