Splunk Cheat Sheets

Quick reference cards for common SPL commands, configurations, and operations — print these out!

SPL Essentials

Basic Search

index=main error earliest=-24h

Search index for "error" in last 24 hours

Field Filtering

index=main src_ip=10.* | table _time, src_ip, action

Filter by field, display specific columns

Count Events

index=main | stats count by sourcetype

Count events grouped by sourcetype

Time Chart

index=main | timechart span=1h count by status

Events over time by status code

Top Values

index=main | top 10 src_ip

Top 10 source IPs by count

Rare Values

index=main | rare 10 user_agent

10 least common user agents

Stats Functions

Count & Sum

| stats count, sum(bytes) as total_bytes by src

Count and sum with alias

Distinct Count

| stats dc(user) as unique_users by src_ip

Count distinct values

Average & Percentile

| stats avg(response_time), perc95(response_time)

Average and 95th percentile

First/Last Values

| stats first(status) as first_status, last(status) as last_status by session_id

First and last values chronologically

List Values

| stats values(action) as actions by user

Collect unique values into multivalue

Multiple Groupings

| stats count by src, dest, action

Group by multiple fields

Eval Functions

Conditional (If)

| eval severity=if(count>100, "high", "low")

If-then-else logic

Case Statement

| eval tier=case(count>1000,"critical", count>100,"high", 1=1,"normal")

Multiple conditions (1=1 is default)

Coalesce (First Non-Null)

| eval ip=coalesce(src_ip, client_ip, source_address)

Use first non-null value

String Functions

| eval domain=lower(split(email,"@",2))

lower(), upper(), split(), replace()

Math

| eval mb=bytes/1024/1024, pct=round(count/total*100,2)

Division, round(), ceil(), floor()

Time Functions

| eval hour=strftime(_time,"%H"), day=strftime(_time,"%A")

Format time to strings

tstats (Data Model Searches)

Basic tstats

| tstats count from datamodel=Web by Web.src

Count from Web data model

With Filter

| tstats count from datamodel=Web where Web.action="blocked" by Web.src, Web.dest

Filter within data model

Time Span

| tstats count from datamodel=Authentication by _time span=1h

Aggregate by time buckets

Summaries Only

| tstats summariesonly=true count from datamodel=Web

Use accelerated summaries only (faster)

Rename Fields

| tstats count from datamodel=Web by Web.src | rename Web.src as src

Clean up data model field names

Multiple Stats

| tstats count, dc(Web.dest) as targets from datamodel=Web by Web.src

Multiple aggregations

REST API Searches

List Indexes

| rest /services/data/indexes | table title, totalEventCount, currentDBSizeMB

Index inventory

List Saved Searches

| rest /services/saved/searches | table title, disabled, cron_schedule

All saved searches

List Apps

| rest /services/apps/local | table title, version, disabled

Installed apps

List HEC Tokens

| rest /services/data/inputs/http | table name, disabled, index

HEC token inventory

List Users

| rest /services/authentication/users | table title, roles, email

User accounts

Server Info

| rest /services/server/info | table serverName, version, os_name

Splunk server details

Time Modifiers

Relative Time

earliest=-24h latest=now

Last 24 hours

Snap to Hour

earliest=-7d@d latest=@d

Last 7 days, snapped to midnight

Business Hours

earliest=-1d@d+9h latest=-1d@d+17h

Yesterday 9am-5pm

This Week

earliest=-7d@w1 latest=@w1

From Monday to Monday

Last Month

earliest=-1mon@mon latest=@mon

Previous calendar month

Specific Date

earliest="01/15/2024:00:00:00" latest="01/16/2024:00:00:00"

Absolute date range

Configuration Files Quick Reference

props.conf — Parsing & Extraction

[mysourcetype] TIME_FORMAT = %Y-%m-%dT%H:%M:%S.%3NZ # Timestamp format TIME_PREFIX = "timestamp":\s*" # What comes before timestamp MAX_TIMESTAMP_LOOKAHEAD = 40 # Characters to search TZ = UTC # Timezone SHOULD_LINEMERGE = false # Don't merge lines LINE_BREAKER = ([\r\n]+) # Event boundary KV_MODE = json # Key-value extraction mode EXTRACT-myfield = regex(?<field>pattern) # Inline extraction FIELDALIAS-cim = vendor_field AS cim_field # Field alias EVAL-calc = field1 + field2 # Calculated field

transforms.conf — Reusable Transforms

[my_extraction] REGEX = src=(?<src_ip>\d+\.\d+\.\d+\.\d+) [route_to_index] REGEX = (ERROR|CRITICAL) DEST_KEY = _MetaData:Index FORMAT = critical_index [nullqueue_healthcheck] REGEX = healthcheck DEST_KEY = queue FORMAT = nullQueue

inputs.conf — Data Inputs

[monitor:///var/log/app/*.log] disabled = false sourcetype = myapp:log index = application [http://mytoken] disabled = false sourcetype = myapp:json index = application

Common CIM Fields

Data Model Key Fields
Web src, dest, url, http_method, status, action, user, bytes_in, bytes_out
Authentication src, dest, user, action, app, src_user, signature
Network_Traffic src, dest, src_port, dest_port, action, transport, bytes
Endpoint dest, user, process, process_path, parent_process, action
Intrusion_Detection src, dest, signature, severity, category, vendor_product

Troubleshooting Commands

Check Data Arrival

index=myindex earliest=-15m | stats count by sourcetype

Verify recent data ingestion

Field Summary

index=myindex | fieldsummary | where count>0

List all extracted fields

Ingestion Delay

index=myindex | eval lag=_indextime-_time | stats avg(lag)

Average seconds between event and index

Search Errors

index=_internal sourcetype=splunkd log_level=ERROR

Splunk internal errors

License Usage

index=_internal source=*license_usage.log type=Usage | timechart sum(b) as bytes

Daily license consumption

Slow Searches

index=_audit action=search info=completed | where total_run_time>60

Searches taking >60 seconds
